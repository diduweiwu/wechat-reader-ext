### V2.5全新版本，支持丰富的配置，所有配置做持久化，重启刷新浏览器不再丢失~

### 使用说明

- 打开一本书的微信读书界面,点击界面右中方“读”字按钮，然后修改相应配置，再打开“滚屏阅读”到开启即可
- 如果需要后台刷时长，请开启“滚屏阅读”和“自动时长”两个开关即可
